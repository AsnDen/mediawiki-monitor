# Mediawiki monitor

Base mediawiki wikis statistics monitor. Uses mediawiki API to get data.

Allows to view general wiki statistics (mainly from `Special:Statistics`), gives direct links
to main page, some `Special:` pages and `Mediawiki:Common.css` and `Mediawiki:Common.js`.

Shows recent changes for each wiki. For each change there is diff support.
